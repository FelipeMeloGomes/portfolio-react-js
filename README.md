## ⚙️ Instalação dos pacotes essenciais.

Instalar os pacotes:

```
npm install
```

Iniciar o Front-End:

```
npm run dev
```

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

-   HTML e CSS.
-   JavaScript.
-   React.
-   Vite.

## 📚 Conceitos aprendidos no projeto

-   JSON.
-   Responsividade.

-   Framework React.
